# Azure Arc Automation License Assign

PowerShell runbook to manage Azure Arc ESU license assignments for Windows Server 2012 machines and clear ESU properties for non-eligible machines.

## Script
- Azure-Arc-Automation-License-Assign.ps1

## Quick start
1. Review parameters and defaults in the script help.
2. Run with -WhatIf first.
3. Run in Azure Automation with System Assigned Managed Identity.
